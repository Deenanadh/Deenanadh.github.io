CineTrack

This web application allows users to explore a vast collection of TV shows and movies, providing detailed information such as summaries, ratings, release dates, and cast information. Users can search for their favorite titles, view trending content, and get recommendations based on their interests. The app pulls data from an external API to keep the information up-to-date, offering a seamless and interactive experience.


Features:
Search Functionality: Users can search for TV shows and movies by title, genre, or actor.
Detailed Information: Displays key details like plot summaries, ratings, release dates, and cast.
Trending Section: Shows currently popular TV shows and movies.
User Interface: Clean and responsive design optimized for both desktop and mobile devices.
